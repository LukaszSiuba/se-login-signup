# ICSI418-Y Homework 1
This repository includes login.html and signup.html to create two web pages namely Login and Signup and styles.css to style both web pages.
The login page contains the display text and field for a user to input their "User ID" and "Password" with a submit button and a link to the signup page.
The Signup page contains the display text and fields for a user to input their "First Name", "Last Name", "User ID", "Password", with a submit button and a link to the login page.
The Styles.css includes relevant code to properly format both webpages to look consistent, center the form on the page, with clean spacing and alignment. Creating a visually appealing
front-end that is simple and effective.

## How To Run
To run both web pages, clone the repository by running: 
```
git clone https://github.com/LukaszSiuba/se-login-signup.git
```
Once cloned, you can decide to change to the directory by running: 
```
cd se-login-signup
```
Afterwards, you could use a plugin called "Live Server" developed by *Ritwick Dey* which will allow you to create a live server for both .html pages in your browser.
Or, you can go into the file you have cloned this repository and open login or signup which will showcase the page.
